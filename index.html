Jaat Lang is a fun, desi-style programming language inspired by Haryanvi culture.
main.jaat:
tha le a = 10
tha le b = 20
tha le sum = a + b
suna de sum
likh de "Ram Ram Bhai"
Translated to JavaScript:
const a = 10;
const b = 20;
const sum = a + b;
console.log(sum);
console.log("Ram Ram Bhai");
How to Start?
        

npm install jaat-lang

create file with the extention .jaat (say main.jaat) in the root directory.
now run npx jaat-lang main.jaat
